1. What is Wyliodrin?

Wyliodrin is a service that allows you to visually create applications for your _embedded_board_ and control it directly from the browser.

2. Why would I use Wylodrin?

With the Wyliodrin service you can create applications for your _embedded_board_ without having to:

  • Connect your board to a monitor
  • Keyboard and mouse, use SSH to connect to it
  • Know a programming language (C, Java or Python) in detail
  • Search for drivers of sensors etc

3. How do I use Wyliodrin?

To start using the service you have to sign up here: www.wyliodrin.com

After signing up please follow the steps at: https://projects.wyliodrin.com/wiki/boards_setup/arduinogalileo

4. How much does it cost to use Wyliodrin?

If you own a Intel Galileo then you are entitled to a one year subscription that allows you to use 3 boards and create 15 applications.

After your subscription has passed, there are multiple types of accounts you can activate, depending on how you want to use the platform:

  • Free account: use one board and create 3 applications
  • Developer account (3.90$/m): use 10 boards and create 20 applications
  • Ultimate account (50$/m): use 250 boards and create 250 applications

5. In which countries is Wyliodrin available?

Wyliodrin is available worldwide.

6. Can I try Wyliodrin without signing up?

Yes, you can. However, you have to acknowledge the fact that the projects you create with that account are public and are deleted on a daily basis.

7. Which are the boards Wyliodrin supports?

For now, Wyliodrin supports Raspberry Pi and Intel Galileo.

8. Do I have to pay to use your tutorials?

No, the tutorials are available to anyone free of charge. You can find them at https://projects.wyliodrin.com/wiki/

9. Can I damage my board if I use Wyliodrin?

Using Wyliodrin cannot damage your _embedded_board_, however, the misuse of electronic parts connected to the board might. That is independent of the platform and Wyliodrin takes no responsibility for any damage arising from such situations.

10. Who can use Wyliodrin?

Wyliodrin can be used anybody, from engineers to artists, hobbyists and even children.

11. What is Visual Programming?

Visual Programming allows you to create applications without having to write the code yourself. You just drag and drop your desired pre-defined application components (blocks) to create the application you want.

12. Where are the programs I write stored?

The programs are stored in the Cloud and can only be accessed by logging in on the Wyliodrin website.

13. Who can see my applications?

The applications are not public. They can be visualised and edited only if you log in. However, applications created as an anonymous user (without signing in) are public.

14. How is my work saved?

The applications are automatically saved as you work on them.

15. Can I download my applications locally, on my computer?

Yes, the projects can be downloaded.

16. Can I run two applications at the same time?

Yes, you can run multiple applications at the same time.

17. My _embedded_board_ does not appear online. What should I do?

First of all, try powring off and on the board, sometimes it happens that the board does not connect due to network issues. If the board is connected to the Internet via WIFI, try connecting it via Ethernet. If that does not work, configure the Intel Galileo board and mark the “Pass Firewalls” option this time.

18. Should I mark the Pass Firewalls option when adding a new _embedded_board_?

No. However, if the board does not connect, add it again to your account and mark the option.

19. My signals do not appear on the graph. Why?

Make sure the name of the signal as used in the “Send signal” application component is the same with the signal on the graph. If they still do not show up, open a shell to your Intel Galileo board and type

redis-cli
flushall
exit

20. How can I pay?

You can pay by using a PayPal account or, if you prefer bitcoins, by using a BitPay account.

21. What is a visual programming block?

A block is an application component used to create Visual Programming applications. Each element you drag and drop to build your application counts as a block.

22. What is a graph?

A graph is an element that allows you track the values received from the sensors. Each element you drag and drop on your dashboard is a graph.

23. My application that uses Facebook does not work. Why?

For most of the applications that use Facebook components, you need an access token. That token has a certain availability. It is possible the token you used has expired, try generating a new one.

24. Why is my promotional Intel Galileo account available for only one month?

The Galileo account is available for one year, but it will renew every month if you keep on using your Intel Galileo board. If you stop using it for one whole month, your special accpunt will be deactivated.
